Les déchets

Madagascar est connu pour ses montagnes d’ordure. On ne peut sortir dans la rue sans voir des déchets partout ; car ils sont omniprésents dans l’environnement de l’île surtout dans la capitale. On les retrouve dans les canaux d’évacuation, les égouts, les rivières dans la plupart des quartiers. Mais en réalité que signifie déchets ?

Le déchet est défini comme tout substance ou tout objet dont le détenteur se défait ou dont il a l’intention ou l’obligation de se défaire.

Classification des déchets

Il y a différents types de déchets selon ses propriétés :

Les déchets dangereux : ce sont les déchets dangereux pour l’humanité et l’environnement comme les déchets inflammables, déchets toxiques, …

Les déchets non dangereux : il s’agit des déchets que l’on peut recycler comme les déchets en verre, ou en plastique, les biodéchets, les bois …, de plus la valorisation de ces déchets créent des milliers d’emplois et de nouveaux produits.
